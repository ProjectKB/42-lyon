## LE PROJET EN LUI MÊME

Pour ce projet, il nous était demandé de refaire les exercices les plus importants de notre piscine (l'examen d'entrée de l'école).

Il permettait de nous introduire aux notions suivantes :

* Commandes Shell de base
* Algorithmie de base
* Pointeur
* Récursivité
* Manipulation de chaîne de caractères
* Manipulation des arguments du main (argv, argc)
* Allocation dynamique
* Directives de préprocesseur et macros
* Pointeur sur fonction
* Makefile
* Header
* Lecture de fichier

Pour plus d'informations, lire le <a href="resources/piscine_reloaded.fr.pdf">sujet</a>.
